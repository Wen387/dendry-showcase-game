# Dendry Showcase Game

A Dendry game scaffold for [Dendry Mod Studio](https://github.com/Wen387/DendryModStudio)'s Featured Templates catalog.

Based on [Social Democracy: An Alternate History](https://github.com/aucchen/social_democracy_alternate_history) by Autumn Chen (MIT License).

## Usage

This project is designed to be downloaded and opened from within Dendry Mod Studio's Welcome Hub under **Featured Templates**. You can also clone it directly:

```bash
git clone https://github.com/Wen387/dendry-showcase-game.git
```
